Selectors Advanced

As we have seen, the three most basics selectors are:

  • Tag names
  • The class attribute
  • The id attribute

Selectors can nevertheless be more complex, that is:

  • They can be combined, for example p.important strong refers to all strong elements inside a p with class important.
  • They can specifiy the DOM structure of the elements, for example div > p will apply only to p that are child (and not descendant) of a div
  • They can target all kind of attributes, not only class and id, and use advanced criteria. For example a[href^="http://"] will match all links that starts with http://, that is, all external links (note the use of the ^).
