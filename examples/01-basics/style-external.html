Style with external file

The best way to style a page is to associate one or more CSS files to the document. For that you need to:

  1. Create or use an existing file with the .css extension (e.g. style.css)
  2. Add a reference to the file(s) in the head of the document using the link tag with appropriate attributes.
  3. For example: <link rel="stylesheet" href="style.css" type="text/css">
  4. The href attribute works exactly in the same way as for links or images, which means it can take both absolute or relative paths.
  5. The CSS file must contain only CSS syntax.
