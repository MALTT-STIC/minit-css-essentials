Block vs. Inline

With CSS you can change the display properties of elements, which are usually either:

  • block: they occupy all the available space and represent a "new line".
  • inline: they occupy just the space they need on the line.

Sometimes it can be useful to make block elements inline or vice-versa. For instance, we can use inline lists elements to build an horizontal navigation menu:

  • About
  • Projects

On the other hand, we can display inline elements as block, for instance if we want buttons to occupy all the available space.
